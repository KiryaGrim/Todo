#Todo Приложение Lab2

##Обзор
Это приложение разработано с использованием WPF (Windows Presentation Foundation) и C#.

##Возможности
Окна Auth, Registration и MainEmpty
Навигация между окнами
Placeholders для полей вводы
Класс для проверки полей ввода

##Использованные технологии
- **C#**: Основной язык программирования.
- **WPF**: Фреймворк для создания настольного приложения.
