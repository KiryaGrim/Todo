# Todo Приложение Lab1

## Обзор
Это приложение Todo List, разработанное с использованием WPF (Windows Presentation Foundation) и C#. Оно предоставляет пользователям интуитивно понятный интерфейс для управления задачами и делами. С помощью этого приложения вы можете легко добавлять, редактировать и удалять задачи, устанавливать приоритеты и сроки выполнения, а также отслеживать прогресс выполнения своих дел. Приложение также поддерживает возможность создания категорий для лучшей организации задач, что делает его идеальным инструментом для повышения продуктивности как в личной, так и в профессиональной жизни.

## Возможности
Окна Auth и Registration
Навигация между окнами
Placeholders для полей вводы

## Использованные технологии
- **C#**: Основной язык программирования.
- **WPF**: Фреймворк для создания настольного приложения.
